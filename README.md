# 🚀 Volunteership Coding Session

Welcome! 👋

This repository contains the code used during a beginner-friendly coding session conducted as part of a volunteering initiative.

The goal of this session is simple:

👉 Show that programming is not magic.
👉 Help beginners write and run their first graphical application.
👉 Spark curiosity about software development.

---

## 🖥️ What You'll Build

In this session, we create a simple desktop application using Python and Tkinter.

Even with just a few lines of code, you can:

* Create windows
* Display text
* Build buttons
* Design graphical interfaces
* Make your own desktop applications

Many students are surprised to learn that software they use every day starts from concepts this simple!

---

## 🎯 Who Is This For?

This repository is for:

* Complete beginners
* Students curious about coding
* Non-CS students exploring technology
* Anyone who has ever wondered:

> "How are apps actually made?"

No prior programming experience is required.

## 🤔 What Is Tkinter?

Tkinter is Python's built-in GUI toolkit.

GUI stands for:

**Graphical User Interface**

Examples:

* Calculator apps
* Music players
* Text editors
* Video players
* Desktop utilities

All of these are built using the same fundamental concepts.
